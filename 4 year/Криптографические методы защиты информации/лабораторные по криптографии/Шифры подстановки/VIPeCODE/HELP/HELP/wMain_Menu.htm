Так выглядит Меню программы:



Главное Меню программы:

Пункт "Файл":

  • "Открыть..." - открыть файл с шифрограммой
  • "Сохранить" - сохранить в файл результат расшифровки
  • "Выход" - завершить работу программы

Пункт "Правка":

  • "Выделить..." - открыть окно "Выделить"
  • "Отменить" - позволяет отменить последнее выполненное действие
  • "Вернуть" - позволяет отменить последнее отмененное действие
  • "Очистить" - сбрасить результат расшифровки

Пункт "Окна":

  • "Шифротекст" - показать/скрыть окно "Шифрограмма"
  • "Алфавит" - показать/скрыть окно "Алфавит"
  • "Открытый текст" - показать/скрыть окно "Открытый текст"
  • "Биграммы" - показать/скрыть окно "Биграммы"
  • "График" - показать/скрыть окно "График"
  • "Информация" - показать/скрыть окно "Информация"

Пункт "Режимы":

  • "Подсветка" - позволяет установить слeдующие режимы подсветки

    • "текущих" - подсветить в окнах все символы совпадающие с текущим
      (находящимся под курсором)
    • "выделенных" - подсветить в окнах все символы совпадающие с выделенным
      (текущий становится выделеннвм после нажатия левой кнопки мыши)
    биграммы
    • "текущих" - подсветить в окнах все биграммы совпадающие с текущей
      (находящейся под курсором в окне "Биграммы")
    • "выделенных" - подсветить в окнах все биграммы совпадающие с выделенной
      (текущая становится выделенным после нажатия левой кнопки мыши)
    -
    • "фразы" - подсветить в окнах все последовательности сочетания букв указанные в окне "Выделить"

  • "Цветовая Гамма" - позволяет выбрать цветовую гамму программы

Пункт "Помощь":

  • "Справка" - данное описание работы с программой.
  • "Home Page" - домашняя страница программы в Интернете.
  • "About" - вывод окна "О программе".
